# 002: 프로젝트 기초 설정 & 좌우 분할 UI 레이아웃

## What to build

프로젝트의 기본 파일 구조(index.html, app.js, style.css)를 생성하고, 좌우 분할 레이아웃의 UI 기본틀을 완성합니다.

**좌측**: SMS 오류 메시지 입력 영역 + [파싱], [초기화] 버튼 + 파싱 결과 표시 영역 (읽기 전용)

**우측**: [검색] 버튼 + 검색 결과 테이블 영역

레이아웃은 반응형이며, 텍스트 입력, 버튼, 결과 영역이 명확하게 분리되어야 합니다. Dexie.js 라이브러리도 함께 추가합니다.

## Acceptance criteria

- [ ] `index.html` 생성: 좌우 분할 컨테이너 구조 구현
- [ ] `app.js` 생성: 기본 빈 파일 준비
- [ ] `style.css` 생성: 좌우 분할 레이아웃 스타일 완성
- [ ] Dexie.js 라이브러리 import (CDN 또는 로컬)
- [ ] 좌측 SMS 입력 텍스트 영역 추가
- [ ] 좌측 [파싱], [초기화] 버튼 추가
- [ ] 좌측 파싱 결과 영역 추가
- [ ] 우측 [검색] 버튼 추가
- [ ] 우측 검색 결과 테이블 영역 추가
- [ ] 브라우저에서 레이아웃 확인 (기능 아직 불필요)
